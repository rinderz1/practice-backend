# Practice Backend

Spring Boot backend project built with Java, Spring Boot, PostgreSQL and Maven.

## Technologies
- Java
- Spring Boot
- PostgreSQL
- Maven

## Clone project

```bash
git clone https://github.com/rinderz1/practice-backend.git
```

## Setup Database

Create database:

```sql
CREATE DATABASE cms_db;
```

Update PostgreSQL credentials in:

```plaintext
cms/src/main/resources/application.properties
```

## Run Project

Run:

```plaintext
CmsApplication.java
```

or:

```bash
mvnw.cmd spring-boot:run
```
